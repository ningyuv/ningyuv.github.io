集成开发环境( IDE, Integrated Development Environment )

  • 大都是一键安装，适用于入门学习

Code::Blocks

  • 推荐
  • 在程序运行结束后默认会有自动暂停，不需要额外设置，避免了同学们（哎？怎么闪退了？）的一些困惑
  • 官网版本不自带汉化，小下载站可能有汉化版 : (，网上也有汉化包但是汉化不完全

下载安装

  • 进入 Code::Blocks 下载页 点击 Download the binary release
  • 找到 codeblocks-<version> mingw-setup.exe
  • <version>: 版本号，视情况而定
  • 在该行后方选择下载源 ( FossHUB or Sourceforge.net )
  • 点击进入，等待数秒即可开始下载，下载完成直接安装即可
  • 之后为简单介绍，不想看可以直接跳到入门使用
  • mingw: 编译器，代表此安装包附带编译器，不用再手动配置。后文会详细介绍
  • setup/nosetup: 代表是否为安装版，这里选择了安装版
  • 安装到最后出现，直接OK

  • 文件关联，自行选择是否需要

MinGW

安装配置

  • 上文提到的编译器，可供编辑器调用
  • 软件官网不好找下载，这里选用 nuwen 的发行版
  • 点击 mingw-<version>-without-git.exe 下载
  • <version>: 版本号，视情况而定
  • without-git 体积小了一半，用到 git 再单独安装
  • 打开安装包，选择一个目录安装，假设该目录为 %MINGW_HOME%
  • 设置系统环境变量，将 %MINGW_HOME%\bin 目录添加到 PATH 中

  • 打开 cmd 输入 gcc --version 正确输出版本信息即配置完成

入门使用

  • 写好一个源文件，假设名为 <filename>.c
  • 在其目录中打开 cmd
  • 输入 gcc <filename>.c -o <filename>.exe
  • 即可编译并生成可执行程序，打开exe即可运行

  • 注意如果双击打开，默认情况下程序运行完毕不会暂停，会直接退出，甚至看不到运行结果
  • 一般不单独使用命令行，会使用编辑器来调用 gcc/g++
  • gcc 可用于编译 C 语言源码，g++ 可用于编译 C++ 语言源码，两者都在 %MINGW_HOME%\bin 目录中
